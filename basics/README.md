# Shell Basics 

This project includes a collection of Bash scripts designed to practice fundamental commands, basic shell navigation, and file manipulation techniques.

0-current_working_directory
Prints the absolute path of the current working directory.

1-listit
Displays the contents of the current directory.

2-bring_me_home
Changes the working directory to the user's home directory.

3-listfiles
Lists files in the current directory using the long format.

4-listmorefiles
Lists all files, including hidden ones, in long format.

5-listfilesdigitonly
Shows a long listing where user and group IDs are displayed numerically.

6-firstdirectory
Creates a directory named my_first_directory inside /tmp.

7-movethatfile
Moves the file betty from /tmp/ to /tmp/my_first_directory/.

8-firstdelete
Deletes the file betty from /tmp/my_first_directory/.

9-firstdirdeletion
Deletes the directory my_first_directory located in /tmp.

10-back
Switches the working directory to the previous directory.

11-lists
Displays, in long format and including hidden files, the contents of the current directory, the parent directory, and /boot.

12-file_type
Prints the file type of /tmp/iamafile.

13-symbolic_link
Creates a symbolic link named __ls__ that points to /bin/ls in the current directory.

14-copy_html
Copies all .html files to the parent directory only if they do not already exist there or if the local version is newer.

15-lets_move
Moves all files beginning with an uppercase letter to the directory /tmp/u.

16-clean_emacs
Removes all files in the current directory that end with ~.

17-tree
Creates the directory structure: welcome/, welcome/to/, and welcome/to/school/.
